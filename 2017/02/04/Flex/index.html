Flex

发表于 2017-02-04

Flex

Flex之前用什么?

  • normal flow
  • float + clear
  • position relative + absolute
  • display inblock-inblock
  • 负 margin

Flex 来了

块级元素垂直排列,行内元素水平排列.
flex 则与方向无关.
flex 空间自动分配,自动对齐.
flex 适用于简单的线性布局,更复杂的布局交给 grid 布局(请期待下个版本)

flex不翻译为弹性布局,因为之前通过百分比的方式的布局已经被成为弹性布局.

Flex 的属性

Flex container 的属性

  • flex-direction 方向
    • row(默认): 主轴为水平方向,起点在左端.
    • row-reverse: 主轴为水平方向,起点在右端.
    • column: 主轴为垂直方向,起点在上沿.
    • column-reverse: 主轴为垂直方向,起点在下沿.
      demo
  • flex-wrap 换行
    • nowrap(默认值):不换行.
    • wrap: 换行,第一行在上方.
    • wrap-reverse: 换行,第一行在下.
      demo
  • flex-flow 上面两个元素的简写
    flex-flow: <flex-direction> || <flex-wrap>;
  • justify-content 主轴方向对齐方式
    • flex-start(默认值): 左对齐.
    • flex-end: 右对齐.
    • center: 居中.
    • space-between: 两端对齐,item之间的间隔都相等
    • space-around: 每个item两侧的间隔相等.所以,项目之间的间隔比项目与边框的间隔大一倍
      demo
  • align-items 侧轴方向对齐方式
    • flex-start: 侧轴起点对齐.
    • flex-end: 侧轴终点对齐.
    • center: 侧轴中点对齐.
    • baseline: item的第一行文字的基线对齐.
    • stretch: 如果item未设置高度或设为auto,将占满整个容器的高度.
      demo
  • align-content 多行/列内容对齐方式
    • 极少用到.

Flex item 的属性

  • flex-grow (剩余空间多时,剩余空间如何分配)定义项目的放大比例.默认为0,值越大分配到的空间越多.
  • flex-shrink(空间不够时,item如何瘦身)定义项目的放大比例,默认为0.值越大,瘦身程度越大.
  • flex-basis(一般不用) 默认占用大小.不写的话就是原始大小
  • flex 上面三个的缩写. flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]. 建议优先使用这个属性.
  • order 顺序 有了这个就可以不用双飞翼了.通过CSS来控制顺序.
  • align-self 允许单个item有与其他item不一样的对齐方式,可覆盖align-items属性.默认为auto,表示继承父元素的align-items属性.如果没有父元素,则等同于stretch.
