发表于 2017-03-19

写在入职之前

明天是入职第一天.
希望作为程序员生涯的第一份工作有个不错的开始.
简历和面试表现被人肯定,真的是件很开心的事情.

学习

目前在线跟着大厂出身的架构师(学霸+学神)学习node.js.
工作将用到react和webpack .
技术上的学习将要三开.
同时学习三个不简单的新东西.
哪怕公司基本不加班,我也必须额外拿出大量的时间去学习.

所以用当初考研 997 的状态去工作和学习吧.

仔细想想
作为单身还没什么交际的程序员的我来说,工作和考研也差不多吧.而且能创造价值,还有薪水拿.

计划清单

  • 2017部分
    1. 打通 react 与 webpack 两项技术.(工作要用到的.权重第一! 进度:刚刚开始)
    2. 学习node.js.通读node文档(刚刚写出了个简单的本地静态资源服务器)
    3. review之前写过的所有jQuery插件.该重构的重构(目前重构了一个)
    4. 把写代码的习惯从ES6迁移到ES5(目前会用新的变量声明方式,解构赋值,箭头函数,反引号….)
    5. promise 虽然明白但是写过的次数比较少.赶紧巩固(加急! 加急! 加急! 加急! )
    6. 整理博客,把之前的一些草稿补完.并且今后坚持一周两篇.
    7. 今天看到了有人在浏览器用js写出了机器学习的围棋AI. 有时间需要研究下. 趁着脑子里考研时学的数学没忘光.
    8. 今年写10W行代码(目前大概写了5000行)
    9. 看十本书或文档(图解HTTP已经看了一半.ES6,node的文档需要看,css揭秘需要看完)
    10. 减掉20斤!!!!!*

中期计划

  1. 学习基本的算法和数据结构
  2. 用 canvas 和node.js 做出个能让五六十人同时使用玩”你画我猜” 的游戏. 过年的时候就可以让大家在姥姥家玩了.
  3. 把简单的C语言复习复习
  4. 给我娘买辆车.让她退休后天天有玩的.(赶紧为这个攒钱)

长期规划

  1. 接触C++.为进一步学习算法做准备.
  2. 学习用Python写个简易爬虫
  3. 增长技术!!!

为美好的世界献上祝福!!!!!!
